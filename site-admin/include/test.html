Place a DIV, first thing, on the main page. z-index:999; width:100%; height:100%; display:none; position:absolute; top:0; left:0; If you feel like getting fancy, you could set the alpha transparency of the div to like 50 percent or something. Then when you open the other window, first show this div. You will have issues with select boxes and other buggy things, ... the easiest way to deal with them is to set them to display:none while the child window is open.
